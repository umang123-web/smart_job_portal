Smart Job Portal

A full-stack web application for job listings, applications, and recruiter management. Built with React, Node.js + Express, and MySQL.

Features
Candidate

Browse jobs with pagination

View job details (description, skills, company profile)

Apply for jobs with resume upload

Save jobs locally (if not logged in) or in DB (if logged in)

Search and filter jobs by title, location, skills, experience

Track application status

Recruiter

Post and manage jobs

View applications in a table

Filter, sort, and export applications to CSV/Excel

Preview candidate resumes

Track candidate status (Applied, Shortlisted, Rejected)

Dashboard charts (bonus: applications per job, status distribution)

Common

Authentication & role-based access (JWT)

Secure password hashing

Responsive UI

Tech Stack

Frontend: React, React Router, Context API or Redux, Vite

Backend: Node.js, Express.js, MySQL2 (Promise-based), Multer for file uploads

Database: MySQL

Authentication: JWT

Styling: TailwindCSS / CSS Modules (optional)
